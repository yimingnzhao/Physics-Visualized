# Physics Visualized

Physics Visualized is a MIT Scratch based project that allows users to interact with basic physics topics that are taught in a high school physics course. 


Topics include:
- Kinematics
- Dynamics
- Energy & Momentum
- Circular Motion
- Simple Harmonic Motion
- Astrophysics
- Thermodynamics
- Fluids
- Optics
- Electricity
- Circuits
